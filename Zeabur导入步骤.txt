🌐 Zeabur导入步骤（详细图文指导）
═══════════════════════════════════════════════════════════

Zeabur网站应该已在浏览器中打开：https://zeabur.com

───────────────────────────────────────────────────────────
步骤1：登录Zeabur（30秒）
───────────────────────────────────────────────────────────

1. 在Zeabur首页，点击右上角 "Sign In" 或 "登录"

2. 推荐使用GitHub登录（最方便）：
   → 点击 "Continue with GitHub"
   → 在弹出窗口中授权Zeabur访问您的GitHub账号
   → 自动登录成功

3. 如果已经登录过，会直接进入控制台

───────────────────────────────────────────────────────────
步骤2：创建新项目（1分钟）
───────────────────────────────────────────────────────────

1. 登录后会看到Dashboard（控制面板）

2. 点击 "New Project" 按钮（蓝色大按钮）

3. 在弹出窗口中：

   Project Name（项目名称）：
   → 输入：ai-tools-hub
   → 或保持默认也可以

   Region（区域选择）⭐ 重要：
   → 选择：Hong Kong (HKG) 🇭🇰
   → 这是离中国最近的区域，访问速度最快

   Environment（环境）：
   → 保持默认：Production

4. 点击 "Create" 按钮

5. 创建成功后会进入项目页面（空白的）

───────────────────────────────────────────────────────────
步骤3：添加Git服务（1分钟）
───────────────────────────────────────────────────────────

1. 在项目页面，看到一个大的加号或 "Add Service" 按钮
   → 点击它

2. 在弹出的选择框中，选择：
   → "Git" 选项（通常有GitHub图标）

3. 会显示您的GitHub仓库列表：

   找到并点击：
   → shihaowei49-cloud/ai-tools-hub

   如果看不到这个仓库：
   → 点击 "Configure GitHub App"
   → 授权Zeabur访问您的仓库
   → 返回后重新选择

4. 选择仓库后，点击 "Deploy"

───────────────────────────────────────────────────────────
步骤4：等待自动部署（1-2分钟）
───────────────────────────────────────────────────────────

Zeabur会自动执行以下操作：

1. 🔍 检测项目类型
   → 识别为 Node.js 项目

2. 📦 安装依赖
   → 运行 npm install
   → 安装 express、cors 等依赖

3. 🚀 启动服务
   → 运行 npm start
   → 启动Express服务器

4. ✅ 部署完成
   → 状态变为 "Running"（绿色）

您可以在页面上看到：
• Building... （构建中）→ 黄色
• Running （运行中）→ 绿色 ✅

如果失败会显示：
• Failed （失败）→ 红色
  → 点击 "Logs" 查看错误信息

───────────────────────────────────────────────────────────
步骤5：生成访问域名（30秒）
───────────────────────────────────────────────────────────

部署成功后（状态显示为 Running）：

1. 在服务卡片上，点击 "Networking" 标签
   → 或者点击服务名称进入详情页

2. 在Networking页面，点击：
   → "Generate Domain" 按钮（蓝色）

3. Zeabur会自动生成一个域名，格式类似：
   → ai-tools-hub-xxx.zeabur.app
   → 或 xxx-ai-tools-hub.zeabur.app

4. 域名生成后（约5秒），会显示在页面上

5. 点击域名链接，或复制到浏览器打开

───────────────────────────────────────────────────────────
步骤6：验证部署成功（1分钟）
───────────────────────────────────────────────────────────

打开生成的域名后，检查：

✅ 页面能正常加载
✅ 看到 "🤖 AI工具集成平台" 标题
✅ 看到8个分类按钮
✅ 看到32个AI工具卡片
✅ 点击分类按钮，筛选功能正常
✅ 点击右下角 "💬 DeepSeek助手"，聊天窗口打开
✅ 在聊天窗口输入问题，DeepSeek能回复

如果所有功能都正常，恭喜！部署成功！🎉

───────────────────────────────────────────────────────────
🎨 可选步骤：绑定自定义域名
───────────────────────────────────────────────────────────

如果您有自己的域名（如 yourdomain.com）：

1. 在Networking页面，点击 "Custom Domain"

2. 输入您的域名，例如：
   → ai-tools.yourdomain.com

3. Zeabur会显示需要添加的DNS记录

4. 去您的域名DNS管理后台添加CNAME记录：
   → Name: ai-tools
   → Type: CNAME
   → Value: xxx.zeabur.app（Zeabur提供的）

5. 等待DNS生效（5-30分钟）

6. 访问您的自定义域名

───────────────────────────────────────────────────────────
❓ 常见问题排查
───────────────────────────────────────────────────────────

Q1: 部署状态一直是 "Building"？
→ 正常情况，第一次部署需要1-2分钟
→ 如果超过5分钟，查看Logs寻找错误

Q2: 部署失败，显示 "Failed"？
→ 点击 "Logs" 查看具体错误
→ 常见原因：
  • package.json配置错误
  • 依赖安装失败
  • 端口冲突

Q3: 网站打不开或显示404？
→ 确认域名已生成
→ 等待几分钟让DNS生效
→ 尝试清除浏览器缓存

Q4: DeepSeek助手无法回复？
→ 确认API Key配置正确
→ 检查浏览器控制台是否有错误
→ 确认网络能访问api.deepseek.com

Q5: 网站访问很慢？
→ 确认选择了Hong Kong区域
→ 检查本地网络连接
→ 考虑升级Zeabur套餐
