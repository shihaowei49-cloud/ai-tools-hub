## 🚀 快速开始

### 1. 打开应用

只需双击 `index.html` 文件，即可在浏览器中打开应用。

### 2. 配置DeepSeek API

1. 点击右下角的"💬 DeepSeek助手"按钮打开聊天窗口
2. 点击"⚙️ API配置"展开配置面板
3. 输入您的DeepSeek API Key（已预填充，您可以修改）
4. API Key会自动保存在本地浏览器中

### 3. 开始使用

- **浏览工具**：在主页面查看所有AI工具，点击分类按钮筛选
- **访问工具**：点击工具卡片上的"访问工具"按钮，在新窗口打开工具网站
- **咨询助手**：在DeepSeek聊天窗口中提问，获取工具推荐和使用建议

## 🆘 常见问题

### Q: DeepSeek助手无法回复怎么办？
A: 请检查：
1. API Key是否正确配置
2. 网络连接是否正常
3. 浏览器控制台是否有错误信息

### Q: 可以添加更多AI工具吗？
A: 可以！编辑 `index.html` 文件，在相应分类下添加新的工具卡片即可。

### Q: 聊天记录丢失了？
A: 聊天记录保存在浏览器的LocalStorage中，如果清除了浏览器缓存会导致记录丢失。

### Q: 在手机上可以使用吗？
A: 完全可以！页面采用响应式设计，在手机和平板上都有良好的体验。
